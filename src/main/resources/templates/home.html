How it works

Domain
Top-level organizer. Groups documents by theme and sets the default chunking strategy for ingestion.
Document
A source file (PDF or EPUB) inside a domain. Tracks ingestion status and the tags assigned at upload time.
Tag
A domain-scoped hierarchical label like cuisine.italian or technique.fermentation. Chunks inherit the tags of their document for filtered retrieval.
Chunk
A text segment produced during ingestion. Each chunk stores a vector embedding, a full-text search index, and denormalized domain and tag metadata so searches need no joins.
Chunking Strategy
Controls how documents are split. Token splits by token count. Semantic splits on meaning shifts. Structural uses document structure (e.g. recipe sections). Set a default per domain and override per document.
